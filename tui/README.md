# Buddy TUI

Terminal UI for Buddy using OpenTUI + the A2A JS SDK.

## Setup

```bash
bun install
```

## Run

Start the Buddy server from the repo root:

```bash
uv run buddy server
```

Then start the TUI from this directory:

```bash
bun dev
```

## Configuration

- `TUI_SERVER_URL` (default: `http://localhost:10001/a2a`)
